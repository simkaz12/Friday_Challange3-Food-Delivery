### 🏃 Run locally

Would like to run this project locally? Open terminal and follow these steps:

1. Clone the repo
   ```sh
   git clone https://github.com/simkaz12/Friday_challange3-Food-Delivery.git
   ```
2. Install NPM packages
   ```sh
   npm i
   ```
   or
   ```sh
   npm install
   ```
3. Run the server
   ```sh
   npm run dev
   ```
